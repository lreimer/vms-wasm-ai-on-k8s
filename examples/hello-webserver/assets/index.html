Beyond Microservices

Running VMs, WASM, and AI Workloads on Kubernetes

About This Repository

This repository demonstrates how to run Virtual Machines (VMs), WebAssembly (WASM), and AI workloads on Kubernetes. It provides examples and configurations to help you get started with these technologies.

Key Features

  • VMs in Action: Learn how to deploy and manage virtual machines on Kubernetes using KubeVirt.
  • WASM in Action: Explore running WebAssembly workloads with Spin and KWasm operators.
  • AI Workloads in Action: Deploy and interact with AI models using tools like Ollama.

Getting Started

Follow the instructions in the README to set up your environment and try out the examples provided in this repository.
